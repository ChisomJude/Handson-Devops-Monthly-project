# BEGINNER: Deploy a static website using Terraform and Basic AWS services


### Project: Personal Portfolio Site Infrastructure

#### What You'll Build:
- Static website hosted on AWS S3
- CloudFront CDN for content delivery
- Route53 for custom domain
- Basic infrastructure as code with Terraform
- Learning Outcomes:
- Understanding IaC fundamentals
- Basic AWS services (S3, CloudFront, Route53)
- Terraform basics (resources, variables, outputs)
- Basic bash scripting for deployment
- Version control with Git

### Tech Stack:
- Terraform
- AWS (S3, CloudFront, Route53)
- HTML/CSS (simple static site)
- Bash scripting
- Git/GitHub

### Implementation Steps:
- Week 1: Setup AWS account, install Terraform, create S3 bucket manually
- Week 2: Write Terraform code to provision S3 bucket with static website hosting
- Week 3: Add CloudFront distribution via Terraform
- Week 4: Create bash script to deploy site updates, add basic monitoring


### Deliverables:
- Terraform configurations with proper structure
- Simple bash deployment script
- Documentation in README
- Live static website URL
